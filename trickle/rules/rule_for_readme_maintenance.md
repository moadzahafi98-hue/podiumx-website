When updating the PodiumX website project

- Always check if the README.md in trickle/notes needs to be updated
- Update README.md when:
  - New pages are added
  - Pricing changes
  - Schedule changes
  - Features are added or removed
  - Contact information changes
  - Any significant structural changes occur
- Keep the README concise and up-to-date
- Include the last updated date in format YYYY-MM-DD
