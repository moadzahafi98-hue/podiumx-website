# PodiumX Performance Center Website

## Overview
Static multi-page marketing website for PodiumX Performance Center – an elite strength, conditioning, and recovery facility in Casablanca, Morocco.

## Brand Identity
- **Name**: PodiumX Performance Center
- **Tagline**: "Elevate Your Performance • Every Level, Every Session"
- **Colors**: Matte black (#000000), Electric green (#00FF5E), White (#FFFFFF)
- **Style**: Minimalist, bold, athlete-focused aesthetic

## Website Structure

### Pages
1. **Home (`index.html`)** – Hero, benefits, facility overview, testimonials, pricing teaser, newsletter
2. **About (`about.html`)** – Story, coaching staff, facility highlights
3. **Programs (`programs.html`)** – Individual coaching tiers, group training, recovery add-on, membership inclusions
4. **Schedule (`schedule.html`)** – Weekly timetable, booking request form
5. **Facility (`facility.html`)** – Zone breakdown, equipment & member experience, tour CTA
6. **Pricing (`pricing.html`)** – Membership plans, recovery add-on, special offers
7. **Blog (`blog.html`)** – Featured articles, community stories, newsletter archive CTA
8. **Contact (`contact.html`)** – Contact details, map embed, message form, partnership CTAs
9. **FAQ (`faq.html`)** – Expandable FAQs, support CTA
10. **Login (`login.html`)** – Member portal placeholder with demo login feedback

### Shared Elements
- Sticky global navigation with responsive mobile menu
- Consistent footer with contact details, hours, and social links
- CSS variables and shared utility classes located in `assets/css/styles.css`
- Reusable JavaScript interactions in `assets/js/main.js`

## Technical Stack
- HTML5 multi-page architecture (no build step required)
- Vanilla CSS with CSS custom properties and responsive layouts
- Lightweight vanilla JavaScript for navigation, sliders, forms, and FAQ toggles

## Interactive Features
- Mobile navigation toggle
- Testimonial slider with auto-advance
- Newsletter subscription feedback state
- Booking, contact, and login forms with inline confirmation messages (non-submitting demo)
- FAQ accordion with animated indicators
